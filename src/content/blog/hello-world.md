## 为什么开始写博客

在 2026 年的今天，AI 可以轻松生成大段文字、分析复杂代码，但我依然相信**人类写作的价值**。写作不是单纯的输出，它是把碎片化的想法理清楚的过程。很多时候，我以为自己懂了某个东西，直到要写下来时才发现到处都是模糊地带。

这篇博客的搭建本身就是一次学习——从选择技术栈到部署上线，每一步都在教我新的东西。

## 技术选型

最终选择了 **Astro + Tailwind CSS + Giscus** 这个组合：

- **Astro**：专为内容型网站设计，默认输出零 JavaScript，构建速度极快。作为刚入门的前端框架，它的学习曲线很友好。
- **Tailwind CSS**：用 utility class 写样式，不用在文件间切来切去，暗色模式支持也很成熟。
- **Giscus**：基于 GitHub Discussions 的评论系统，免费、无需后端，天然支持 Markdown。

## 接下来的计划

1. 每周至少写一篇技术笔记
2. 把大学期间零散的笔记整理迁移过来
3. 逐步自定义主题和排版，让这个角落更有"自己的感觉"

这个博客不追求完美，追求持续。Let's go。
